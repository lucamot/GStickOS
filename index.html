Welcome to GStickOS

A lightweight, ready-to-use retro gaming system.

Download GStickOS

GStickOS is a custom firmware/image for Game Stick 4K Lite that allows you to play retro games using RetroArch. The system is designed to boot directly into RetroArch, providing a seamless gaming experience.

Features:

  • Boots directly into RetroArch for an instant gaming experience.
  • Supports external USB storage for adding game ROMs.
  • Includes around 50 Libretro cores, with the ability to add more.
  • SSH server enabled for remote access.
  • Supports WiFi connectivity via an external USB WiFi dongle.
  • No ROMs or copyrighted materials are included.

How to install:

  1. Download GStickOS image and unzip it.
  2. Flash the image to an SD card with Win32 Disk Image or similar.
  3. Put the SD on your Game Stick Lite 4K and start playing!
