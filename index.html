Welcome to GStickOS

A lightweight, ready-to-use retro gaming system.

Coming soon...

GStickOS is a custom firmware/image for Game Sticks 4K Lite that allows you to play retro games using RetroArch. The system is designed to boot directly into RetroArch, providing a seamless gaming experience.

Features:

  • Boots directly into RetroArch for an instant gaming experience.
  • Supports external USB storage for adding game ROMs.
  • Includes around 50 Libretro cores, with the ability to add more.
  • SSH server enabled for remote access.
  • Supports WiFi connectivity via an external USB WiFi dongle.
  • No ROMs or copyrighted materials are included.
